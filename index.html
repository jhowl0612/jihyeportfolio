git bash

  • git init : 현재 폴더를 로컬 저장소로 지정
  • git add 폴더이름/파일이름 : 해당 폴더/파일을 대기상태로 올림
  • git commit -m "설명" : 대기상태의 대상을 커밋상태로 올림. 설명서를 붙임
  • git remote add origin https://github.com/jihye0612/jihyeportfolio.git : 해당 링크의 저장소를 'origin'이라는 이름으로 연결. 연결저장소로 지정.
  • git push -u origin master : 커밋 상태의 대상들을 원격저장소 origin으로 옮김. 이후부터는 git push만 입력해도 됨.
  • git rm --cached -r 폴더이름/
