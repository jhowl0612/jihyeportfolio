메소드로 날짜와 시간 표시하기

getYear() 년도 (1970년 이후의 년도)
getFullYear() 년도를 구합니다
getMonth() 월을 구합니다
getDate() 일을 구합니다
getDay() 요일을 구합니다
getHours() 시간을 구합니다
getMinutes() 분을 구합니다
getSeconds() 초를 구합니다
getTime() 1970년 1월 1일 이후 시간을 1000분의 1초로 표시
getMilliseconds() 100분의 1초로 표시

사용예) D-Day 표시하기

<head>
    <title>MY Birthday</title>
    <script language="JavaScript">
        var today = new Date();
        var birth = new Date("June 1, 2002");
        // Janu Febr Marc Apri May June July Augu Sept Octo Nove Dece
        var aa = birth.getTime();
        var bb = today.getTime();
        var day = Math.ceil((bb-aa)/24/60/60/1000);
    </script>
</head>

<body>
    <img src="cup.gif" width="377" height="286" style="display: block; margin: 0 auto; border:0;">

    <script language="JavaScript">
        document.write("<p style='text-align: center;'>")
        document.write("내가 지구상에 태어난지 " + day + "일 지났어요.")
        document.write("</p>")
    </script>
</body>
